# 🏢 Enterprise IT Infrastructure & Network Design

This repository contains the proposal, design, and simulation for a complete IT infrastructure and network upgrade for a film and media company. This project was completed collaboratively as part of the "Hardware and Networking (CIS1103)" course.

## 🚀 Project Overview

The company's previous hardware was outdated and underpowered. The objective of this project was to design a modern, cost-effective IT solution (within a 38,000 AED budget) to replace outdated computers, provide mobility for the film crew, invest in dedicated servers, and upgrade the network to support both wired and wireless connections.

### Key Features & Deliverables:
- **Hardware Procurement:** Selected high-performance workstations for the Graphics Editor, standard PCs for Administration, and mobile laptops for the Film Crew.
- **Server Provisioning:** Planned and allocated resources for dedicated Email and File servers.
- **Network Architecture:** Designed a comprehensive network layout combining wired and wireless connections.
- **IP Addressing Scheme:** Developed a structured IP subnetting plan for efficient network routing.
- **Security Policy:** Proposed strict network security requirements and implementation policies.
- **Network Simulation:** Built and tested a fully functional prototype of the network using Cisco Packet Tracer.

## 🛠️ Technologies & Tools Used

- **Cisco Packet Tracer:** For network simulation, configuration, and testing.
- **IT Infrastructure Planning:** Hardware sourcing, budgeting, and resource allocation.
- **Network Protocols:** IP Addressing, Subnetting, Routing, and Wireless configurations.

## 💡 Roles & Contributions

As a core member of this collaborative project, my responsibilities included:
*   **Hardware Research:** Assisting in researching and selecting the appropriate hardware components and servers that meet the company's technical requirements and budget.
*   **Network Design & Simulation:** Collaborating on the network layout and helping build the simulated network topology using Cisco Packet Tracer.
*   **Documentation:** Contributing to the final report, specifically in detailing the network requirements and configuration testing.
